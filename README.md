# Simple React Modal

import { Modal } from 'modal-react';

# Props

- `children` - The modal content.
- `isVisible` - Shows the modal.
- `backdropColor ` - Backdrop/mask color.
- `onBackdropClick` - Event is called on backdrop/mask click.
